# Rails Sidenotes — JetBrains Plugin

A JetBrains IDE plugin that reads sidecar YAML annotation files generated by the [sidenotes](https://github.com/1stvamp/sidenotes-ruby) Ruby gem and displays ActiveRecord schema information directly in your editor.

Compatible with **RubyMine** and **IntelliJ IDEA** (with the Ruby plugin).

## Features

### Gutter Icons
A database icon appears next to `class User < ApplicationRecord` declarations. Click it to see a schema summary popup showing table name, column count, indexes, and associations.

### Quick Documentation (Ctrl+Q)
Hover over or press Ctrl+Q on a model class name to view the full schema as formatted HTML — columns with types, nullability, defaults, indexes, and associations.

### Inlay Hints
Column type information appears inline next to attribute references like `self.email`, showing hints such as `string, not null`. Toggle these in **Settings → Editor → Inlay Hints → Sidenotes Schema Hints**.

### Sidenotes Tool Window
A dedicated panel (default: right sidebar) displays the full schema for whichever model file is currently open. Automatically updates when you switch editor tabs.

### Regenerate Annotations
Run `bundle exec rake sidenotes:generate` directly from **Tools → Regenerate Sidenotes Annotations**. Progress and results are shown via IDE notifications.

### Auto-Refresh
A VFS file watcher monitors the `.annotations/` directory. When annotation files are created, modified, or deleted, the cache updates automatically — no restart needed.

## How It Works

The plugin looks for YAML files in `.annotations/` relative to your project root. File mapping follows Rails conventions:

| Model file | Annotation file |
|---|---|
| `app/models/user.rb` | `.annotations/user.yml` |
| `app/models/admin/user.rb` | `.annotations/admin/user.yml` |

Annotation YAML files are generated by the `sidenotes` gem and contain structured data:

```yaml
---
table_name: users
primary_key: id
columns:
  - name: id
    type: integer
    nullable: false
  - name: email
    type: string
    nullable: false
    default: ""
indexes:
  - name: index_users_on_email
    columns: [email]
    unique: true
associations:
  - type: has_many
    name: posts
    class_name: Post
    foreign_key: user_id
```

## Installation

### From Disk
1. Download the latest `.zip` from [Releases](https://github.com/wesmason/rails-sidenotes-jetbrains/releases)
2. In your IDE: **Settings → Plugins → ⚙️ → Install Plugin from Disk...**
3. Select the downloaded `.zip` file
4. Restart the IDE

### Build from Source
```bash
git clone https://github.com/wesmason/rails-sidenotes-jetbrains.git
cd rails-sidenotes-jetbrains
./gradlew buildPlugin
```

The plugin `.zip` will be in `build/distributions/`.

## Requirements

- IntelliJ Platform 2024.1 or later
- RubyMine or IntelliJ IDEA with the Ruby plugin
- The [sidenotes](https://github.com/1stvamp/sidenotes-ruby) gem installed in your Rails project

## Development

Open the project in IntelliJ IDEA and run:

```bash
./gradlew runIde        # Launch a sandboxed IDE with the plugin loaded
./gradlew buildPlugin   # Build the distributable .zip
./gradlew verifyPlugin  # Run IntelliJ Plugin Verifier
```
